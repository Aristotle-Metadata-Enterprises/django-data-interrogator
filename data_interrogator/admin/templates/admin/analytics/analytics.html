How to make a table

  1. Select a model (table) to generate queries from
  2. Filters can be entered with any of the following:
    = / != equal to / not equal to
    > / >= greater than / greater than or equal to
    < / <= less than / less than or equal to
    For example: age > 10.
  3. Filters, Columns, and Sort by can be any field on a model, or using dot-separated syntax to query across. For example:
    age age of the base query object
    parent.age age of the object linked by the related parent object
    parent.sibling.age age of the sibling of the parent object
  4. Sort by can be reversed by prepending a minus symbol: -
